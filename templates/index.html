Observation Zone App Compatibility

Zones are saved in CUP format with R1/A1/R2/A2 fields. Compatibility when sharing via different formats:

Preset Params SeeYou XCSoar (CUP) LK8000 (CUP) XCTSK QR
Cylinder A1=180°, R2=0 Cylinder CylinderZone CIRCLE ✓ Full
FAI Sector A1=45°, R1=3km, R2=500m FAI Sector CustomKeyhole DAe sector ⚠ Outer circle only
BGA Keyhole A1=45°, R1=10km, R2=500m BGA Keyhole BgaKeyholeZone DAe sector ⚠ Outer circle only
BGA Fixed Course A1=45°, R1=20km, R2=500m BGA Fixed Course BGAFixedCourse DAe sector ⚠ Outer circle only
BGA Enhanced Option A1=90°, R1=10km, R2=500m BGA Enhanced BGAEnhancedOption DAe sector ⚠ Outer circle only
Start Line Line, A1=90°, R2=0 Start Line LineSectorZone LINE ⚠ Circle (ignored)
Finish Line Line, A1=90°, R2=0 Finish Line LineSectorZone LINE ✓ LINE (LK8000)

⚠ XCTSK QR v2 supports sector angle (A1) and line flag only — inner radius (R2) is not transmitted. All apps use CUP file for full fidelity.
